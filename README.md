# toc4pdf.tocgen
Repository of manually entered tables of content (TOCs).

```sh
pdftocio -o out.pdf doc.pdf < toc
```
See https://github.com/Krasjet/pdf.tocgen/blob/master/README.md for the details of adding TOCs to PDF files.
